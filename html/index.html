Welcome to the PocketBase Management Interface
This server manages multiple PocketBase instances using Docker Compose.

Available Commands

./pbc.sh list - List all projects and their status

./pbc.sh add <projectname> [port] - Add a new project

./pbc.sh remove <projectname> - Remove a project

./pbc.sh start <projectname|all> - Start a project or all projects

./pbc.sh stop <projectname|all> - Stop a project or all projects

./pbc.sh cleanup - Clean up orphaned containers

Access Your Projects

Projects are accessible via:

  • Subdomain: http://<projectname>.angusjs.xyz
  • Path-based: http://pb.angusjs.xyz/<projectname>/
